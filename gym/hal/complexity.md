

# Searching
* Linear Search O(N)
* Binary Search O(log_2 N)
* Tertiary Search O(log_3 N)

# Sorting
* Bubble sort O(N^2)
* Selection Sorting O(N^2)
* Insertion Sorting O(N^2)
* Merge Sorting O(NlogN) 
* Quick Sorting O(N^2) to O(NlogN)
* Counting Sorting O(N+K)
* Radix Sorting `O( (n+b)*log_b (max x))`
		`O(n*log_b(n))`
* Heap Sort O(logN)
* Bucket Sort O(N)
