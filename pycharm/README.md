# PyCharm 

## Installation on Ubuntu 20.04 
```
cd ~/Downloads
wget https://download.jetbrains.com/python/pycharm-community-2020.3.1.tar.gz
tar -xzf pycharm-community-2020.3.1.tar.gz
cd pycharm-community-2020.3.1/bin/
chmod u+x pycharm.sh
sh pycharm.sh
```
