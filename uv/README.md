# uv
> An extremely fast Python package and project manager, written in Rust.

## Installation
```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

## Test
```bash
uv venv --python 3.12
source .venv/bin/activate
```

## Remove 
```bash
rm -rf .venv
```
